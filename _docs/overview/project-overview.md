# Project Overview

## What is KishEditor?

**KishEditor** is a modern, client-side LaTeX editor built with React and TypeScript. It provides a comprehensive editing experience for LaTeX documents with both **source code editing** and **WYSIWYG (What You See Is What You Get)** modes, making it accessible to both LaTeX experts and beginners.

## Key Characteristics

### Client-Side Architecture
- **No server required**: Everything runs in the browser
- **Privacy-focused**: Your documents never leave your device
- **Fast**: No network latency for editing operations
- **Offline-capable**: Works without an internet connection

### Dual Editing Modes
- **Source Mode**: Direct LaTeX code editing with syntax highlighting
- **WYSIWYG Mode**: Visual editing with formatting toolbar
- **Seamless Switching**: Switch between modes without losing content

### Real-Time Preview
- **Live Rendering**: See your LaTeX rendered as you type
- **Split-Pane View**: Edit and preview side-by-side
- **Debounced Updates**: Optimized performance for large documents

## Target Use Cases

1. **Academic Writing**: Research papers, theses, and academic documents
2. **Mathematical Content**: Documents with complex equations and formulas
3. **Technical Documentation**: Technical manuals and documentation
4. **Educational Content**: Teaching materials and course notes
5. **Blog Posts**: Articles that require mathematical notation
6. **Embedded Editors**: Integration into existing web applications

## Technology Stack

### Core Technologies
- **React 18+**: Modern React with hooks
- **TypeScript**: Type-safe development
- **Vite**: Fast build tool and dev server

### Editing Libraries
- **CodeMirror 6**: Source code editor with LaTeX syntax highlighting
- **Tiptap (ProseMirror)**: WYSIWYG editor framework
- **MathLive**: Visual math equation editor

### Rendering
- **KaTeX**: Fast math rendering engine
- **LaTeX.js**: LaTeX document structure parsing

## Project Goals

1. **Accessibility**: Make LaTeX editing accessible to non-experts through WYSIWYG mode
2. **Performance**: Fast, responsive editing experience
3. **Extensibility**: Easy to customize and extend
4. **Integration**: Simple to integrate into any React project
5. **Type Safety**: Full TypeScript support

## What Makes KishEditor Different?

### vs. Traditional LaTeX Editors
- **Visual Editing**: WYSIWYG mode eliminates the need to know LaTeX syntax
- **Modern UI**: Clean, intuitive interface
- **React Integration**: Built for modern web applications

### vs. Online LaTeX Editors
- **Privacy**: All processing happens client-side
- **No Account Required**: Works immediately without signup
- **Offline Support**: Can work without internet connection
- **Customizable**: Full control over styling and behavior

### vs. Desktop LaTeX Editors
- **Web-Based**: Access from any device with a browser
- **No Installation**: No need to install LaTeX distribution
- **Collaborative Ready**: Can be integrated into collaborative platforms
